VOD Archive - How to Use
========================

For Obliviosa:

1. Double-click "Start VOD Archive.bat"

2. The first time only: It will install a small dependency (takes ~30 seconds).
   After that, it starts instantly.

3. Your browser will open automatically with the VOD archive.

4. To stop: Close the black window that says "VOD Archive" (or press Ctrl+C in it).


If it says "Node.js is not installed":
- Go to https://nodejs.org
- Download the "LTS" version (green button)
- Install it (just click Next through the steps)
- Then try double-clicking the batch file again

That's it! The VOD list updates automatically from the web each time you open it.
